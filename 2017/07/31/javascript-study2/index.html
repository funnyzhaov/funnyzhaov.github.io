数据类型

  • Number :表示整数和浮点数

  • NaN

    1.即非数值(Not a Number) 是一个特殊的数值、运算式本来应该返回一个数值，但结果不是数值

    2.typeof(NaN)–>number

    3.任何涉及NaN的操作(例如NaN/10)都会返回NaN

    4.NaN和任何值都不相等，包括NaN本身

  • isNaN(n)

    功能：检测n是否是”非数值”、返回值boolean

    说明：isNaN()对接收的值，先尝试转换为数值，再检测是否为非数值

数值转换

  • Number():用来强制转换数据类型，可用于任意类型

  • parseInt() ：操作的参数必须以数字开头，否则将无法转换,结果为NaN

    说明：

    1.parseInt()会忽略字符串前面的空格，直至找到第一个非空格字符。

    2.parseInt()这个函数提供第二个参数：转换时使用的基数(即多少进制8/10/16)

  • parseFloat() 专门用于把字符串转换成数值

    说明：

    1.从第一个字符串开始解析每个字符，直到遇到一个无效的浮点数字为止
