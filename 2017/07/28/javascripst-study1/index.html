JavaScript是整个前端的核心，之前一直以为会用CSS写出炫酷的网页就是前端高手了，现在看来，非也、非也啊~

web前端和Android移动端，甚至服务端都有相似之处，也就是现在大家都在推崇的敏捷开发，配置好框架，专心写业务，写交互就够了。

HTML中导入外部标签

注释：

// 单行注释

/**/ 多行注释

分号：

每行语句结束时如果不加分号，则有解释器自动添加
标识符

1.区分大小写
ECMAScript中的变量，函数名，操作符都区分大小写

2.标识符的命名规则：

(1).由字母、数字、下划线(_)或美元符号($)组成

(2).不能以数字开头

(3).不能使用关键字、保留字等作为标识符

变量

1.变量类型：由变量的值来决定，不需要声明变量类型

2.变量声明语法：

//声明后再赋值

//多个变量声明(声明的同时赋值)

注：1.省略var声明的变量是全局变量

2.不推荐省略var操作符来定义全局变量

数据类型

简单数据类型

  • Undefined

  • Null

  • Boolean

  • Number

  • String

undefined:

undefined类型只有一个值，即特殊的undefined.
一般而言，不存在需要显示地把一个变量设置为undefined值的情况.

null:

null表示一个空对象指针

如果定义的变量准备在将来用于保存对象，那么最好将该变量初始化为null，而不是其他值。

undefined值是派生自null值的，所以undefined==null的返回结果是true.

复杂数据类型:

object

typeof

功能：检测变量类型

语法：typeof 变量 /typeof(变量)

返回值:string类型，有可能的值：string、number、boolean、object、
undefined、function
